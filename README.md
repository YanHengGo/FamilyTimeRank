# FamilyTimeRank

家族全員の **SNS利用時間を可視化** し、  
**制限ではなく“気づきと会話”を生む** ための iOS アプリです。

> ❌ 制限・ブロックしない  
> ⭕ 家族全員が同じ結果を見る  
> ⭕ ランキング形式で毎日レポート通知

---

## 📱 コンセプト

FamilyTimeRank は、  
「スマホ使いすぎ」を **管理・監視** ではなく  
**見える化・共有・会話** によって改善することを目的としています。

- 親も子も同じ土俵に立つ
- 数字で見て気づく
- 家族の会話のきっかけを作る

---

## ✨ 主な機能（MVP）

- 📊 家族全員の **SNS利用時間ランキング**
- 🕒 1日・1週間単位の利用時間確認
- 🔔 毎日同じ内容のレポート通知を家族全員に送信
- 👨‍👩‍👧‍👦 家族グループ作成・招待コード参加
- 🔒 プライバシー重視設計
  - 取得するのは「SNSカテゴリの合計時間」のみ
  - 個別アプリ名・内容は取得しない

---

## 🧭 ユーザーフロー概要

### 初回起動（家族作成者）
1. アプリ起動
2. 家族グループ作成
3. 招待コード発行
4. 利用時間取得の同意
5. ホーム画面へ

### 初回起動（参加者）
1. アプリ起動
2. 招待コード入力
3. 表示名設定
4. 利用時間取得の同意
5. ホーム画面へ

### 毎日の利用
- アプリ起動 or 通知確認
- 今日のランキングを確認
- 必要なら週間レポートを見る

---

## 🔔 レポート通知例

【今日のSNS利用ランキング】

1位 パパ：3時間20分
2位 ママ：1時間45分
3位 お兄ちゃん：1時間10分
4位 妹：0時間50分

家族平均：2時間01分

※ 家族全員に **同じ内容** が届きます。

## 🔐 プライバシー設計

- 取得する情報

- 取得しない情報
  - 個別アプリ名
  - メッセージ内容・閲覧内容
- データ共有範囲
  - 家族グループ内のみ
- いつでも共有停止可能
